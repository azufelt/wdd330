  • Advantages/Disadvantages of Libraries‚Äî
    • Advantage:
      • used by lots of people and throoughly tested
      • High browser functionality
    • Disadvantages:
      • Must include library code, on top of your own code. Potential performance issues.
      • May have to change the functionality of the library, which can require some DIGGING through code.
      • Could make code slower than using vanilla JavaScript.
    • When to use a library‚Äî You have to learn the library's notation, which can be difficult! Will likely already be cached if you use a CDN. Popularity swings could shift quickly "here today, gone tomorrow". Some libraries just have too much stuff, and much is not very commonly useful.
